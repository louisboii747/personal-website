# LouisDev Personal Website

A modern, responsive portfolio website built to showcase my projects, systems, and experiments.

Live site: https://louisdev.uk

---

## Overview

This site is my personal hub on the web, a place to highlight what I build, what I’m learning, and what I’m working on.

It’s designed with a clean, modern aesthetic and focuses on performance, simplicity, and real-world deployment.

---

## Features

* Modern dark UI with gradients and smooth animations
* Fully responsive (works on mobile, tablet, desktop)
* Fast global delivery via CDN
* HTTPS enabled (secure by default)
* Interactive terminal-style section
* Live stats system (expandable with backend integration)
* Automatic deployment via GitHub

---

## Tech Stack

* **HTML5**
* **CSS3**
* **Vanilla JavaScript**
* **Cloudflare Pages** (hosting & CDN)
* **GitHub** (version control & CI/CD)

---

## Deployment

This project is deployed using **Cloudflare Pages**:

* Push to `main` branch
* Site automatically builds and deploys
* Served globally via Cloudflare CDN

---

## Domain & SSL

* Custom domain configured: `louisdev.uk`
* SSL provided via Cloudflare (Full Strict mode)
* Automatic HTTPS enforcement enabled
